## Quickstart

To access the dashboard, follow these steps:

1. **Start the application**: Run `docker-compose up` to start the application.
2. **Access the dashboard**: Open your web browser and navigate to `http://localhost:5000`.
3. **Log in**: Use the default credentials provided in the `app.py` file to log in.

For more detailed instructions, refer to the [architecture.md](architecture.md) file.

To get started with Isomer, follow these steps:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/your-repo/isomer.git
   cd isomer
   ```

2. **Build the Docker containers**:
   ```bash
   docker-compose build
   ```
   ```bash
   docker-compose build
   ```

3. **Start the application**:
   ```bash
   docker-compose up
   ```

4. **Access the dashboard**:
   Open your web browser and navigate to `http://localhost:5000` to access the Isomer dashboard.

5. **Log in**:
   Use the default credentials or create a new user to log in.

For more detailed instructions, refer to the [Architecture](#architecture) and [Configuration](#configuration) sections.
